Introduction

This website is currently work-in-progress!

These are things in JavaScript which make us sad. Inspired by PHP Sadness, we aim to archive all the ugly JavaScript warts.
If you want to contribute sadnesses, start a pull request on our github repository.

JavaScript is an old, but living and evolving language. As a consequence, it is possible that some sadnesses are going to be resolved over time.
If you want to notify us that a certain sadness has been resolved, start an issue at our github repository.
